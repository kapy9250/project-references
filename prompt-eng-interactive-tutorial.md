## 项目简介
Anthropic 官方推出的互动式 Prompt Engineering（提示词工程）教程。该课程旨在提供一个全面的、循序渐进的指南，帮助用户掌握如何在 Claude 模型中设计出最优的提示词。通过本课程的学习，用户可以掌握编写好提示词的基本结构、识别常见的失败模式并学会应用“80/20”法则解决它们、理解 Claude 模型的优缺点，并能够从零开始为常见用例构建强大的提示词。

## 核心特性
- **分级课程结构**：课程内容分为初级、中级和高级（包含 9 个章节），以及超前技巧附录，循序渐进地引导学习。
- **互动式游乐场 (Example Playground)**：每节课底部都有实践区域，用户可以自由调整提示词并观察 Claude 回应的变化，加深直观理解。
- **丰富的真实案例与练习**：除了理论讲解外，还配备了各行业的复杂案例（如法律、金融、编程等领域的提示词构建练习）以及相应的答案解析。
- **涵盖前沿技巧**：内容不仅包括基础的“分离指令与数据”、“赋予角色”等，还深入到避免幻觉、预见性思维（思维链）、复杂提示词构建、工具使用以及检索增强（RAG）等进阶领域。

## 使用场景 (或 核心概念 / 架构)
- **初学者入门**：学习基础的 Prompt 结构、如何做到清晰直接，以及如何为 AI 分配角色。
- **进阶优化**：学习如何通过思维链（Thinking Step by Step）、少量样本（Few-shot prompting）和输出格式控制来提升输出质量。
- **工业级应用落地**：对于需要在实际业务（如构建客服 Chatbot、金融分析辅助、法律文档处理等）中接入大模型的开发者，提供直接可借鉴的复杂 Prompt 构建范式。
- *备注：该教程主要基于轻量级、快速的 Claude 3 Haiku 模型运行，但也完全适用于更强大的 Sonnet 和 Opus 模型。*

## 适用人群
- **AI 开发者与工程师**：需要将 LLM 集成到应用中并提升稳定性的技术人员。
- **产品经理与业务人员**：希望通过提示词技术快速验证 AI 落地场景的非技术人员。
- **大模型研究者与爱好者**：想要深入理解 Claude 行为逻辑、探索前沿 Prompt 技巧的用户。
