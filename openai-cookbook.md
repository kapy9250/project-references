# OpenAI Cookbook

**GitHub**: https://github.com/openai/openai-cookbook

**Stars**: 71,872 ⭐ | **Forks**: 12,080

## 项目简介

OpenAI Cookbook 是 OpenAI 官方维护的示例和指南仓库，涵盖了使用 OpenAI API 的各种最佳实践和高级用法。它是目前最受欢迎的 OpenAI 学习资源之一。

## 内容分类

### 1. 基础指南
- API 认证与入门
- 文本生成 (Completions)
- 聊天补全 (Chat Completions)
- 嵌入 (Embeddings)

### 2. 高级技巧
- **提示工程 (Prompt Engineering)**
  - Chain of Thought (思维链)
  - Few-shot Learning
  - System Prompt 设计
  
- **函数调用 (Function Calling)**
  - 工具使用
  - 结构化输出
  
- **令牌与成本优化**
  - 令牌计算
  - 成本控制策略

### 3. 特定用例
- **代码辅助**
  - 代码审查
  - Bug 修复
  - 代码解释
  
- **数据处理**
  - 文本分类
  - 情感分析
  - 实体提取
  
- **内容创作**
  - 文章写作
  - 营销文案
  - 邮件生成

### 4. 集成指南
- 与 LangChain 集成
- 与 LlamaIndex 集成
- 与 Pinecone 向量数据库集成
- 与 Slack、Discord 等平台集成

### 5. 企业级应用
- 安全与合规
- 多租户架构
- 成本优化
- 性能调优

## 特色内容

### API Guides
详细的 API 使用指南，覆盖所有 OpenAI 端点。

### Jupyter Notebooks
大量可运行的 Jupyter Notebook 示例，可直接下载运行。

### 使用案例
来自真实场景的应用案例和解决方案。

## 学习路径建议

1. **入门**: 阅读 `getting_started` 目录
2. **进阶**: 学习 `prompting_guides` 和 `fine_tuning`
3. **实战**: 参考 `use_cases` 中的案例
